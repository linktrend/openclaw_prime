# Product layer

Per-repository product truth: **grounding**, **programs**, and **reports**.

- **grounding/** — vision, ship criteria, intent, constraints, glossary
- **programs/** — active program trees (`<program-id>/PROGRAM.md`, modules, phases, issues)
- **reports/** — agent reports per issue (empty until execution)

Factory bootstrap history stays under `LiNKdev/factory/programs/bootstrap/` (not here).
