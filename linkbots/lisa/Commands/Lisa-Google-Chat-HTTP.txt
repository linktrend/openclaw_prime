--------------------------------------------------------------------
You do NOT need Google Workspace Marketplace admin, and you do NOT
install from Apps admin. Use Google Chat "Find apps" only.

1. Open Google Chat in a browser while signed in as calusa@linktrend.media
   → https://chat.google.com
2. In the left sidebar, look for "Apps" (or "Direct messages" / chat list).
3. Click the + (plus) next to Apps / "Find apps" / "Browse apps"
   (wording varies slightly; goal is the app finder / marketplace for Chat).
4. In the search box, type exactly:   Lisa Prime
   - Do NOT search only "Lisa" (the old broken app was named that).
   - You should see "Lisa Prime" with description like
     "Lisa Prime - LiNKtrend assistant".
5. Click Lisa Prime → Chat / Install / Add (whatever the button says).
6. Open the new 1:1 chat with Lisa Prime and send:  hi
7. If Lisa asks for pairing approval, tell the Mini operator to run:
       PATH="/opt/homebrew/opt/node@24/bin:$PATH"
       node ~/Projects/openclaw_prime/openclaw.mjs --profile lisa pairing list
       node ~/Projects/openclaw_prime/openclaw.mjs --profile lisa pairing approve googlechat <CODE>

If you still only see old "Lisa" (not "Lisa Prime"):
  - Hard-refresh Chat (or try an incognito window as calusa@).
  - Confirm you are signed in as calusa@linktrend.media (only that
    address is on the visibility list).
  - Remove/ignore any old "Lisa" DM if it still sits in your chat list
