     https://linktrend-mini.tailf7e13a.ts.net:8443/#token=...

  gateway.controlUi.dangerouslyDisableDeviceAuth=true

  Control UI over the Tailscale Serve URL uses the gateway token instead of repeated
  browser-device pairing. This is acceptable here because the gateway is bound to
  loopback, exposed to Carlos only through tailnet Serve :8443, and the public
  Funnel path is only /googlechat on :443.
